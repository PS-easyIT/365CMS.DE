# Theme Management

**Ordner:** `doc/admin/themes/`

Verwaltung des visuellen Erscheinungsbilds.

## Module

### 1. Theme Übersicht (`README.md`)
- **Datei:** `admin/themes.php`
- Liste aller installierten Themes.
- Aktivieren/Deaktivieren von Themes.
- Live-Vorschau.

### 2. Theme Editor (`EDITOR.md`)
- **Datei:** `admin/theme-editor.php`
- Visueller Editor für Theme-Optionen (Farben, Typografie, Layout).
- Bearbeitung von Templates und CSS-Dateien direkt im Browser (für erfahrene Nutzer).
