# Dashboard Widgets
**Datei:** `admin/design-dashboard-widgets.php`

Anpassen der Admin-Startseite (`admin/index.php`).

## Funktionen

### 1. Sichtbarkeit
- Ein-/Ausblenden von Standard-Widgets:
  - "Auf einen Blick"
  - "Schnellentwurf"
  - "WordPress Nachrichten" (optional)
  - "System Status"

### 2. Sortierung
- Festlegen der Reihenfolge der Widgets.
- Spalten-Layout (1, 2, 3 oder 4 Spalten).

### 3. Custom Widgets
- Hinzufügen eigener HTML-Widgets (z.B. für Support-Infos oder Notizen).
- Rollenbasierte Sichtbarkeit (nur für Admins sichtbar).
