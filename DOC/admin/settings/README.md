# General Settings

**Datei:** `admin/settings.php`

Globale Konfiguration des CMS.

## Bereiche

### 1. Allgemein
- Seitentitel und Untertitel.
- Admin-E-Mail-Adresse.
- Sprache und Zeitzone.
- Datums- und Zeitformat.

### 2. Schreiben & Lesen
- Standard-Kategorie für Beiträge.
- Startseite (Statische Seite oder letzte Beiträge).
- Anzahl der Beiträge pro Seite.

### 3. Diskussion
- Kommentar-Einstellungen (Erlauben, Moderation, Blacklist).
- Avatar-Einstellungen.

### 4. Permalinks
- URL-Struktur (Pretty Permalinks).
- Basis-Pfade für Kategorien und Tags.
