# Design Management

**Ordner:** `doc/admin/design/`

Hier werden alle administrativen Funktionen zur optischen Gestaltung verwaltet.

## Module

### 1. Schriftarten (`FONTS.md`)
- **Datei:** `admin/fonts-local.php`
- Verwaltung lokaler Google Fonts (GDPR-konform).
- Upload eigener Webfonts (.woff, .woff2).

### 2. Menüs (`MENUS.md`)
- **Datei:** `admin/menus.php`
- Drag & Drop Menü-Builder.
- Verwaltung mehrerer Menü-Positionen (Header, Footer, Sidebar).

### 3. Dashboard Widgets (`DASHBOARD-WIDGETS.md`)
- **Datei:** `admin/design-dashboard-widgets.php`
- Anpassen der Widgets auf dem Admin-Dashboard.
- Aktivieren/Deaktivieren von Modulen.
