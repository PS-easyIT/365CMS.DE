# Beispiel-Assets außerhalb von `CMS/uploads`

Dieses Verzeichnis enthält versionierte Beispiel- und Referenzdateien, die **nicht** als produktive Laufzeit-Uploads im Runtime-Baum unter `CMS/uploads/` liegen sollen.

## Regel

- `CMS/uploads/` bleibt für echte Laufzeitdaten reserviert.
- Versionierte Demo-, Mock- oder Referenzdateien gehören nach `DOC/assets/examples/` oder in einen anderen expliziten Doku-/Fixture-Pfad.
