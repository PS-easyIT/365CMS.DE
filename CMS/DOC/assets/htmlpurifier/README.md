# HTMLPurifier

## Kurzbeschreibung

`HTMLPurifier` bereinigt HTML-Ausgaben sicher und schützt vor unsauberen oder gefährlichen Inhalten.

## Quellordner

- `CMS/assets/htmlpurifier/`

## Verwendung in 365CMS

- zentrale Nutzung in `CMS/core/Services/PurifierService.php`
- Delegation zusätzlich über `CMS/includes/functions.php`
