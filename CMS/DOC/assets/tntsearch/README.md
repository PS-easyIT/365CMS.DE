# TNTSearch

## Kurzbeschreibung

`TNTSearch` stellt die Volltextsuche von 365CMS bereit.

## Quellordner

- `CMS/assets/tntsearchhelper/`
- `CMS/assets/tntsearchsrc/`

## Verwendung in 365CMS

- direkte Nutzung in `CMS/core/Services/SearchService.php`
- Suchfluss zusätzlich über `CMS/core/Router.php`
