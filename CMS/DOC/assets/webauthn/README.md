# lbuchs/WebAuthn

## Kurzbeschreibung

`lbuchs/WebAuthn` ist die Basis für Passkeys und WebAuthn-Anmeldung in 365CMS.

## Quellordner

- `CMS/assets/webauthn/`

## Verwendung in 365CMS

- direkte Nutzung in `CMS/core/Auth/Passkey/WebAuthnAdapter.php`
- Einbindung über `CMS/assets/autoload.php`
