# Member Dashboard

**Datei:** `member/index.php`

Zentraler Einstiegspunkt für registrierte Mitglieder.

## Funktionen
- **Übersicht:** Aktueller Status, letzte Aktivitäten, Benachrichtigungen.
- **Widgets:** Übersicht über gebuchte Pakete, aktive Projekte oder Tickets (abhängig von installierten Plugins).
- **Navigation:** Zugriff auf Profil, Einstellungen und geschützte Inhalte.
