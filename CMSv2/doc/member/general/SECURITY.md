# Security Center

**Datei:** `member/security.php`

Sicherheitsrelevante Einstellungen für das Mitgliedskonto.

## Funktionen
- **2-Faktor-Authentifizierung (2FA):** Aktivieren/Deaktivieren (TOTP).
- **Login-Historie:** Anzeige der letzten Logins (Zeit, IP, Browser).
- **Aktive Sessions:** Übersicht und Möglichkeit, andere Sitzungen zu beenden.
