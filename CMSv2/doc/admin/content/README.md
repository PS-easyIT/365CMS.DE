# Content Management

**Ordner:** `doc/admin/content/`

Verwaltung aller inhaltlichen Elemente der Website.

## Module

### 1. Landing Page (`LANDING-PAGE.md`)
- **Datei:** `admin/landing-page.php`
- Spezieller Editor für die Startseite.
- Verwaltung von Sektionen (Hero, Features, Testimonials).

### 2. Seiten (`PAGES.md`)
- **Datei:** `admin/pages.php`
- Erstellen und Bearbeiten statischer Seiten (Impressum, Kontakt, Über uns).
- Hierarchische Struktur (Eltern/Kind-Seiten).

### 3. Beiträge (`POSTS.md`)
- **Datei:** `admin/posts.php`
- Blog/News-System.
- Kategorien und Tags.

### 4. SEO (`SEO.md`)
- **Datei:** `admin/seo.php`
- Globale SEO-Einstellungen (Titel-Formate, Meta-Beschreibungen).
- XML Sitemap Generierung.
- Robots.txt Editor.
