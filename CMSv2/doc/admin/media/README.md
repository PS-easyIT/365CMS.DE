# Media Library

**Datei:** `admin/media.php`

Zentrale Verwaltung aller hochgeladenen Dateien (Bilder, Dokumente, Videos).

## Funktionen

### 1. Übersicht
- Grid-View und Listen-Ansicht.
- Filter nach Dateityp und Datum.
- Suche nach Dateinamen.

### 2. Upload
- Drag & Drop Upload-Bereich.
- Automatische Generierung von Thumbnails.
- Unterstützung für WebP-Konvertierung (optional).

### 3. Bearbeitung
- Bild-Editor (Zuschneiden, Drehen, Skalieren).
- Bearbeiten von Metadaten (Alt-Text, Titel, Beschreibung).
