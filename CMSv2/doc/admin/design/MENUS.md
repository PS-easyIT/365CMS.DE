# Menu Management
**Datei:** `admin/menus.php`

Verwaltung der Navigationsstrukturen der Website.

## Funktionen

### 1. Menü erstellen
- Beliebige Anzahl von Menüs anlegen.
- Name vergeben.

### 2. Menüpunkte hinzufügen
- **Seiten:** Aus existierenden Seiten wählen.
- **Beiträge:** Blogartikel verlinken.
- **Individuelle Links:** Externe URLs.
- **Kategorien:** Links zu Kategorie-Archiven.

### 3. Strukturierung
- Drag & Drop Sortierung.
- Verschachtelung (Drop-Down Menüs) durch Einrücken.

### 4. Positionen
- Zuweisung zu Theme-Locations (z.B. "Header Menu", "Footer Menu").
