## 🇩🇪 Deutsch

### ✨ Warum 365CMS?

365CMS verbindet klassisches CMS-Content-Management, Mitgliederbereiche, SEO-/Performance-Werkzeuge, Theme-Customizing, Plugin-Erweiterbarkeit und Betriebswerkzeuge in einer konsistenten PHP-Architektur. Der Fokus liegt auf **betriebssicherem Self-Hosting**, **klarer Admin-Struktur**, **lokal dokumentierten Workflows** und **sauber integrierbaren Erweiterungen**.

### 🚀 Schnellstart

#### Voraussetzungen

- PHP `8.4+`
- MySQL `5.7+` oder MariaDB `10.3+`
- Apache `2.4+` mit `mod_rewrite`
- PHP-Erweiterungen: `PDO`, `pdo_mysql`, `mbstring`, `json`

> Hinweis: Die produktiv gebündelten Symfony-Komponenten unter `CMS/assets/` (`mailer`, `mime`, `translation`) verlangen PHP 8.4. 365CMS prüft diese Mindestplattform bereits vor dem regulären Bootstrap.

#### Installation

1. Projekt auf den Webserver kopieren.
2. Datenbank-Zugang in `config.php` bzw. der Installationsroutine hinterlegen.
3. Browser öffnen und `install.php` aufrufen.
4. Admin-Konto anlegen.
5. Nach der Installation `install.php` entfernen.

#### Empfohlene Nacharbeiten

- HTTPS und Rewrite-Konfiguration prüfen
- Cache-/Upload-/Log-Verzeichnisse beschreibbar machen
- Mail-Transport konfigurieren
- LDAP optional in `CMS/config/app.php` aktivieren
- Theme auswählen und Theme-Customizer prüfen

### 🚀 Quick Start

#### Requirements

- PHP `8.4+`
- MySQL `5.7+` or MariaDB `10.3+`
- Apache `2.4+` with `mod_rewrite`
- PHP extensions: `PDO`, `pdo_mysql`, `mbstring`, `json`

> Note: The bundled Symfony runtime components inside `CMS/assets/` (`mailer`, `mime`, `translation`) require PHP 8.4. 365CMS validates this platform baseline before the normal bootstrap continues.

#### Installation

1. Copy the project to your web server.
2. Provide database credentials in `config.php` or through the installer.
3. Open `install.php` in the browser.
4. Create the admin account.
5. Remove `install.php` after installation.

#### Recommended follow-up steps

- Verify HTTPS and rewrite configuration
- Make cache, upload and log directories writable
- Configure mail transport
- Optionally enable LDAP in `CMS/config/app.php`
- Select a theme and review the theme customizer
